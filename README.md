## About MyMonsterWar
MonsterWar 是一个使用 Entt, SDL3, glm, ImGui, nlohmann-json 和 Tiled 开发的跨平台 C++ 塔防游戏。

## WebAssembly Demo

Try the live WebAssembly demo:
- Main: [https://mjj1997.github.io/MyMonsterWar/](https://mjj1997.github.io/MyMonsterWar/)
- Develop: [https://mjj1997.github.io/MyMonsterWar/develop/](https://mjj1997.github.io/MyMonsterWar/develop/)

The `main` branch deploys to the root, `develop` to `/develop/`, and tags to `/tagname/`.
